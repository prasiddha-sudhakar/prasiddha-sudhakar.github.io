Logarithms

Using logarithms (In this case, of log(GDP)) are useful, as we will be able to see the whether GDP is increasing, decreasing or constant over time

  1. Growth rate of logarithms
    Let's say that we have an economy that grows at rate y from t-1 to t

    \[y = { y_{t} - y_{t-1} \over \ y_{t-1} }\]

    We can now simplify this whole thing:

    \[1+y = { y_{t} \over \ y_{t-1} }\]

    Recall that log(1+x) = x
    We cam use this rule here: log(yt/yt-1) = y Which is the same as saying log(yt) - log(yt-1) ≈ y
  2. Average growth and logarithms
    Now, let's assume that we have an increase in GDP by 2% each year and the economy is currently (Yo) producing 100 units of real output

    \[{{ y_{t} - y_{t-1} \over \ y_{t-1}} = 0.02}\]

    So that:

    \[{{ y_{t}} = 1.02 * y_{t-1}}\]

    We can use this information to calculate the GDP in years to come, as we have found that overall, the formula is:

    \[{{ y_{t}} = (1.02)^{t} y_{0}}\]

    After taking the logarithm of both sides, we get: log(yt) = log(Yo) + log(1.02)^t
    We can see that this is a the equation of a straight line, and that average growth is constant

Aggregate Production and Productivity

Measuring Productivity

This is done by measuring which country can produce more output from given input

Also known as the production function

Production Function

\[ Y = F(k, l) \]

Where:

  • Y: Output
  • K: Capital
  • L: Labor

Let's take a look at the Cobb Douglas production function

\[ F(k, l) = Ak^bL^{1-b} \]

Where:

  • a: Elasticity
  • b: Total Factor Productivity
      How much output is created for given input k & l

Let's say as an example, what would happen if we double both K and L?

We would have to substitute K with 2K and L with 2L

This would end up giving us:

\[ 2F(K, L) \]

What does it tell us about the output? It doubles

Giving us constant returns to scale

To find output per worker, we can rearrange the equation to get:

\[ Y = A (\frac{ K }{ L })^b \]

Giving us:

\[ y = \frac{ Y }{ L }\]

\[ k = \frac{ K }{ L }\]

To get the per worker production function, it would be:

\[ y = Ak^b\]

Example

In this example, we're going to find out which team is more productive

\[ F(k, l) = Ak^bL^{1-b} \]

\[ y = Ak^b\]

Setting up the Sollow Model

\[ Y = A_tK_t^bL_t^{1-b} \]

Where the t subscript can denote the time

We can also assume that A and L are constant and there is no government expenditure

Per worker variables

We know that:

\[ y_t = \frac{ Y_t }{ L_t }\]

\[ k_t = \frac{ K_t }{ L_t }\]

We can also define:

\[ c_t = \frac{ C_t }{ L_t }\]

\[ i_t = \frac{ I_t }{ L_t }\]

We've also assumed a closed economy where there is no government spending and net exports = 0

\[ y_t = c_t + i_t\]

There are some other factors that we need to consider:

That a fraction of output is saved in each period

\[ i_t = sy_t = sA{k_t}^b\]

And that a fraction of machines wear out each month. This is denoted with δ

\[ d_t = δk_t\]

Evolution of capital stock:

\[ Δk_t = i_t - d_t\]

Where:

\[ k_t = \text{ change in capital per worker}\]

\[ i_t = \text{ investment per worker}\]

\[ d_t = \text{ depreciation per worker}\]

We should be able to use the Solow model to predict outcomes. First, let's visualize this graphically

Start by plotting these curves:

\[ y_t = A{k_t}^b\]

\[ i_t = sA{k_t}^b\]

\[ d_t = δ{k_t}\]

\[ \text{If: }sA{k_t}^b = δ{k_t}\]

\[ k^* = ({ sA \over δ})^{1 \over 1-b}\]

This is also known as the steady state capital-labor ratio

When the kt is small, output and investment are high (See Graph) and depreciation is low

When the kt is large, capital becomes less productive due to diminishing marginal returns

Keep in mind that y* - i* = c*

Where c* is consumption per worker

Output per worker converges to y* = A(k*)^b

Consumption per worker converges to c* = (1-s)y*

Implications

If A is constant, growth will eventually stop due to diminishing marginal returns

And if economies are under their 'steady state' will grow faster than usual and converge to their steady state

Reasons for converging

Simple: Law of diminishing returns

As the economy grows, depreciation increases faster than output and investment -> Economy eventually reaches a new steady state

However, this whole time we haven't considered what would happen if there were an increase in labor force

Population growth in the Solow model

Our new dt equation would look like this:

\[ d_t = (δ+n){k_t}\]

When considering y* and k*, we need to make sure that Yt and Kt are growing at the same rate as Lt

Our new equation to find k* would now be:

\[ k^* = ({ sA \over δ + n})^{1 \over 1-b}\]

This proves that:

  • A higher n would lead to a decrease in k
  • To compensate for a higher n, we would need to increase s

Fundamentally...

The Solow Model predicts that as the number of workers increase, there are fewer machines per worker and each worker produces less

However, the issue is, is that the Solow Model only considers workers in terms of capital instead of new technologies and ideas that they may bring
