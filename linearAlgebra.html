1.1 Matrices and Vectors

Matrix: Rectangular array of numbers

It is essential to use the correct notation, which will be in Upper Case letters

Example:

A = \begin{bmatrix}3 & 0 & 2\\\sqrt{2} & 0 & 1\end{bmatrix}

Whereby, the size of the matrix is:

  • m - rows
  • n - columns

We could call this as a m x n matrix

To target specific rows and columns, we would use the (ith row, jth column) format

Example:

\[a_{2,3} = 1\]

Trow matrices are equal if they have the same size and the corresponding entries are the same

Addition of matrices

If A and B have the same size, A + B is a matrix of the same size with the sum of corresponding entries

Mathematically, this would look like:

\[A_{m*n} + B_{m*n} = C_{m*n}\]

Where:

\[a_{i,j} + b_{i,j} = C_{i,j}\]

Properties of matrix addition and scalar multiplication

Let A, B,C, D be matrices and s, t be scalars

  1. A + B = B + A
  2. A + (B+C) = (A+B) + C

  3. \[A + 0_{m*n} = A\]

  4. \[A + (-A) = 0_{m*n}\]

  5. (st) A = s(tA)
  6. s(A+B) = sA + sB
  7. (s+t)A = sA + tA

The transpose of a matrix

Let's suppose that A is an m*n matrix

\[A^t \text{ is the m*n matrix by switching the row and columns}\]

\[A = \begin{bmatrix}1 & 3 & -5\\2 & 0 & 1\end{bmatrix}\]

\[A^t = \begin{bmatrix}1 & 2 \\3 & 0 \\ -5 & 1 \end{bmatrix}\]

Vectors

  • A row vector is a 1 * n matrix
  • A column vector is a n * 1 matrix

Example:

\[u = \begin{bmatrix}1 \\2 \\ -1\end{bmatrix}\]

Where u is a column vector

1.2 Linear combinations of vectors

\[ \text{Define } R^{n} = {\vec{v}} \text{ in a n*1 matrix}\]

which is the set of n*1 matrices and m dimensional vectors

Suppose that

\[\vec{v_1}, \vec{v_2} ... \vec{v_n} \text{are vectors in } R^{n} \text{(for some m)} \]

A linear combination of these vectors is a vector of the form:

\[c_1* \vec{v_1} + c_2*\vec{v_2} + c_k*\vec{v_k} \]

Where all the c's are scalars

Examples

Let

\[\vec{v_1} = \begin{bmatrix}1 \\2 \\ -1\end{bmatrix}\]

\[\vec{v_2} = \begin{bmatrix}3 \\0 \\ 1\end{bmatrix}\]

\[\vec{v_3} = \begin{bmatrix}0 \\1 \\ 1\end{bmatrix}\]

\[\vec{v_4} = \begin{bmatrix}1 \\0 \\ 0\end{bmatrix}\]

Some combinations are ...

  1. \[2\vec{v_1} + 3\vec{v_3}\]

    \[2\begin{bmatrix}1 \\2 \\ -1\end{bmatrix}\ + 3\begin{bmatrix}0 \\1 \\ 1\end{bmatrix}\ = \begin{bmatrix}2 \\7 \\ -5\end{bmatrix}\]

  2. \[-5\vec{v_4}\]

    \[-5\begin{bmatrix}1 \\0 \\ 0\end{bmatrix}\ = \begin{bmatrix}-5 \\0 \\ 0\end{bmatrix}\]

  3. \begin{bmatrix}0 \\0 \\ 0\end{bmatrix}

    This is the linear combination of v1, v2, v3 and v4
